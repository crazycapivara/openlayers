OpenLayers 4 R

-----

Quick start

Install and load openlayers


      install.packages("openlayers")

      library(openlayers)

Create a new map object


      map <- ol()

Add tiles


      map <- ol() %>%
        add_osm_tiles()

Add some features


      map <- ol() %>%
        add_osm_tiles() %>%
        add_features(geojsonio::us_cities[1:5, ],
                     style = icon_style())
